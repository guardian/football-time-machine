Football Time Machine

The Football Time Machine! allows you to simulate time travel by replaying old PA data.

Usage

You can use the Football Time Machine! by changing the URL you use to access the PA API. Instead of (e.g.) http://pads6.pa-sport.com/ you should point to the Football Time Machine!, including the date and time you wish to travel to. For example if you set --time machine domain--/20140225/1550/ you will recieve data as if it is 15:50 on the 25th of Feb 2014.

You'll need to choose a time and date for which there exists saved data. The following date/time combinations are eligible for time travel

Note, historical data may not work for you if the API calls you are making weren't recorded at the time.
