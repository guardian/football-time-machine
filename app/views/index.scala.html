Football Time Machine

The Football Time Machine! allows you to simulate time travel by replaying old PA responses.

Usage

You can use the Football Time Machine! by changing the URL you use to access the PA API. Instead of (e.g.) http://pads6.pa-sport.com/ you should point to the Football Time Machine!, including the date and time you wish to travel to. For example if you set http://@request.host/replay/20140225/1550 you will receive data as if it is 15:50 on the 25th of Feb 2014. Not, in Frontend the property to set is called football.api.host.

If you want to record data for future time travel adventures then set you should set the PA API URL to http://@request.host/record. This will transparently save PA API responses into your data directory while returning the full response to your application.

Note, historical data may not work for you if the API calls you are making weren't recorded at the time.
